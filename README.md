# Blood Donation Platform

A C# console app connecting blood donors with people in need. Donors register with their location and blood type, consumers post urgent requests, and the system matches them intelligently.

## Features

- User registration and login for donors and consumers
- Donor profiles with location and blood group details
- Post and track blood requests
- Smart matching based on location and blood type
- Accept requests and manage donations

## How to Run

1. Clone the repository
2. Open in Visual Studio
3. Build and run the application
4. Choose your role (Donor or Consumer) and follow the menu
